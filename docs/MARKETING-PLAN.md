# INKVERSE — แผนการตลาดนอกเว็บ (Track C)

> เป้าหมาย: ดึงผู้อ่านมังงะไทยเข้าเว็บ → สมัคร (รับ 20 เหรียญ) → ติดตอนเบ็ดเกี่ยว → เติมครั้งแรก 2x

## 1. Growth Funnel ภาพรวม
```
โซเชียล/SEO  →  เข้าเว็บ  →  สมัคร (+20 เหรียญฟรี)  →  อ่านฟรี 1–3 ตอน
     →  เจอตอนล็อก (cliffhanger)  →  เติมครั้งแรก 2x  →  เช็คอินรายวัน (retention)
     →  เติมซ้ำ / ปลดล็อกยกเรื่อง (bulk −15%)
```
ทุกช่องทางข้างล่างมีหน้าที่ดันคนเข้า funnel นี้

## 2. ช่องทาง Acquisition (กลุ่มคนอ่านมังงะไทย)

### TikTok (ช่องหลัก — ROI สูงสุดสำหรับคอนเทนต์ภาพ)
- คลิปสั้น "พรีวิวมังงะ" 15–30 วิ ตัดช่วงพีค + ค้างไว้ → "อ่านต่อที่ INKVERSE"
- ใช้นักแปล/ทีมงานเป็นหน้าช่อง สร้างความผูกพัน
- Hashtag: #มังงะ #อ่านมังงะ #การ์ตูน + ชื่อเรื่อง

### X (Twitter) — คอมมูนิตี้มังงะ/อนิเมะไทยแข็งแรง
- โพสต์หน้าปก + 1 ช่องเด็ด + ลิงก์ ตอนมีตอนใหม่
- รีพลายในกระแสอนิเมะ/มังงะที่กำลังดัง

### Facebook Groups / Discord
- กลุ่มการ์ตูน-มังงะไทย: แชร์เรื่องใหม่แบบไม่สแปม (ให้คุณค่าก่อน)
- ตั้ง Discord ของ INKVERSE: แจ้งตอนใหม่, อีเวนต์เช็คอิน, รับฟีดแบ็ก

### เว็บบอร์ด/ชุมชนเฉพาะทาง
- โพสต์รีวิว/แนะนำเรื่องในบอร์ดการ์ตูน พร้อมลิงก์ deep-link ไปหน้าเรื่อง

## 3. กลยุทธ์ "นักแปลคือ Influencer" (ตัวคูณสำคัญ)
ระบบมีโครงนักแปล + ส่วนแบ่ง 80% + ranking อยู่แล้ว → ใช้เป็นกลไกการตลาด:
- **ให้นักแปลโปรโมตงานตัวเอง** — ยิ่งคนอ่าน/ปลดล็อกเยอะ ยิ่งได้เงิน แรงจูงใจตรงตัว
- ทำ **asset สำเร็จรูป** ให้นักแปลแชร์ (การ์ดหน้าปก, คลิปเทมเพลต)
- โชว์ **อันดับนักแปล** (มีหน้า TranslatorRanking แล้ว) เป็น social proof ดึงนักแปลเก่งเข้าระบบ
- Referral seeding: เริ่มจากนักแปลชวนแฟนคลับเดิมของตัวเองเข้ามา

## 4. ต่อยอด SEO (มีฐานแล้ว — ขยายผล)
มีอยู่: `SEO-SETUP.md`, `lib/seo.ts`, `components/seo/JsonLd.tsx`, sitemap
- **หน้า genre/ranking = landing page** ทำ title/description ให้ตรงคีย์เวิร์ด "อ่าน[ชื่อเรื่อง]", "มังงะ[แนว]"
- JSON-LD `Book`/`ComicSeries` ต่อเรื่อง → rich result บน Google
- เนื้อหา evergreen: หน้า "มังงะแนะนำตามแนว" ดึง organic traffic ระยะยาว
- internal linking จากหน้าเรื่อง → ตอน → เรื่องใกล้เคียง

| จ. | TikTok | คลิปพรีวิวตอนใหม่ |

## 6. Referral (เฟสถัดไป — ออกแบบไว้ล่วงหน้า)
- ชวนเพื่อนสมัคร เพื่อนเติมครั้งแรก → ได้เหรียญทั้งคู่
- โตแบบ viral ต้นทุนต่ำ จับคู่กับโบนัสสมัคร 20 + เติมแรก 2x ได้พอดี
- รายละเอียด implement อยู่ใน roadmap ของแผนหลัก

## 7. งบ & การวัดผล
- เริ่มจาก **organic + นักแปล influencer** (งบ ~0) ก่อนยิงแอด
- เมื่อ funnel นิ่ง (activation rate ดี) ค่อยเติมงบ TikTok/Meta ads ไปยังหน้า landing ที่ convert ดีที่สุด
- ติด UTM ทุกลิงก์ → วัดว่าช่องไหนพาคนมาเติมเงินจริง (ดูจาก `CoinOrder` ที่ PAID)
- ตัวเลขเหนือสุดที่ต้องดู: **CAC (ต้นทุน/ผู้จ่ายใหม่)** เทียบ **LTV** — คุ้มเมื่อ LTV > 3×CAC

# 🚀 Launch Playbook (เปิดตัวจริง)

## Pre-launch (ก่อนเปิด)
- ✅ ปิดช่องโหว่การเงิน (ดู SECURITY.md) + ตั้ง Omise live + R2 CORS (ดู GO-LIVE.md)
- เตรียม **คอนเทนต์ตั้งต้น**: เรื่องเด่น 10–20 เรื่อง, ตอนต้นฟรี, ตั้ง "ตอนเบ็ดเกี่ยว" ล็อกช่วงพีค
- ดึง **นักแปล 5–10 คน** พร้อมลงผลงาน (พวกเขาคือ growth engine)
- หน้า legal ครบ (terms/privacy/dmca/about)

## Soft launch (สัปดาห์ 1–2)
- เปิดให้นักแปล + แฟนคลับของพวกเขาเข้าก่อน (referral seeding: ทั้งคู่รับ 50 เหรียญ)
- ตั้ง **Discord** ชุมชน + แจ้งตอนใหม่
- TikTok/X: คลิปพรีวิว 15–30 วิ ตัดช่วงพีค → "อ่านต่อที่ INKVERSE"

## Growth loop (สัปดาห์ 3+)
```
โซเชียล → สมัคร (+20 เหรียญ) → ติดตอนเบ็ดเกี่ยว → เติมครั้งแรก 2x
   → เช็คอินรายวัน + แจ้งเตือนตอนใหม่ (retention) → referral ชวนเพื่อน (viral)
```
- เปิด **VIP** + **verified นักแปล** (เก็บเงินได้) เป็น upsell

## Paid ads (เมื่อ funnel นิ่ง)
- เริ่ม organic + นักแปล influencer (งบ ~0) ก่อน
- ยิงแอด TikTok/Meta ไปหน้า landing ที่ convert ดีสุด **เมื่อ LTV > 3×CAC** เท่านั้น
- ติด UTM ทุกลิงก์ → วัดช่องที่พาคนมาเติมเงินจริง (`CoinOrder` PAID)

## Timeline 30/60/90
- **30 วัน**: soft launch, นักแปล+คอนเทนต์พร้อม, วัด activation (สมัคร→เติมแรก)
- **60 วัน**: เปิด referral/VIP เต็มตัว, เริ่มแอดทดสอบงบเล็ก, ปรับ funnel ตาม KPI
- **90 วัน**: สเกลแอดช่องที่คุ้ม, ระบบ retention ครบ, เริ่มทำ SEO content evergreen

## KPI ที่ต้องดูทุกสัปดาห์
Activation (สมัคร→เติมแรก) · ARPPU · repeat-topup % · coin breakage % · DAU เช็คอิน · CAC vs LTV
